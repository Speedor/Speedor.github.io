Github

        Github的帐户申请和基本使用在这里就不啰嗦了,Github在建博中起最大作用的就是GitHub Pages,简单点儿可以理解为github提供的一个免费的博客后台，你可以把Hexo生成的静态文件deploy到github repository上，然后github就会自动帮你渲染你的博客内容，最重要的是免费，免费，免费，重要的事儿说三遍。 - 创建Github repository：在Github上创建名为.github.io的repository（username就是你Github的用户名，这个是规定哦），之后的博客内容就在这个仓库中管理。 - 配置GitHub Pages：在.github.io的setting->options->GitHub Pages中进行配置主题等主要配置，配置好之后打开https://username.github.io这个链接就能看到pages已经搭建完成。

Hexo

        Hexo相对来说比较陌生一点儿，说白了主要作用就是生成博客静态文件的内容。我们可以选择自己喜欢的theme进行配置，牛逼的你也可以写出自己独特风格的theme，之后就是静态文件的generate和deploy，很简单吧，放心啦这个坑不深-_-
