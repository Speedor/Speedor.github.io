docker常用命令

  • 查看docker已启动容器： docker ps
  • 查看docker所有容器： docker ps -a
  • 查看docker镜像： docker images
  • 启动coinspider容器： docker start coinspider
  • 关闭coinspider容器： docker stop coinspider
  • 进入coinspider容器（默认root用户）： docker attach coinspider

  • 退出coinspider容器： Ctrl+p+q

  • 创建spider容器（以ubuntu:16.04为镜像）： sudo docker run -it --name spider --restart=always -p 5003:5000 -v /home/XinXin/dockerfiles/:/app ubuntu:16.04 /bin/bash

  • 删除spider容器 sudo docker rm spider

  • 创建ubuntu:16.04镜像 docker pull ubuntu:16.04

  • 删除ubuntu:16.4镜像 docker rmi ubuntu:16.04
